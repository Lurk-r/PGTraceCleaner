# Pixel Gun Trace Cleaner - PGTC
(aka PG BanTraceCleaner)

PGTC is a simple script made for reducing ban chances when creating or playing on new PG accounts by cleaning local traces (i.e account/ban traces, logs) left from Pixel Gun 3D/PG3D.

**[DOWNLOAD](https://github.com/Lurk-r/PG-Trace-Cleaner/releases/latest)**

---
## Features

- Detect and wipe PG's temp files + logs.
- Clear registry values contain account/ban traces.
- Find account IDs found under PG local data.
- Your PG settings won't reset.
- Lightweight & portable.

---
## Notes

- Join our Discord for support, guides, tools, more games/apps etc.: **https://dsc.gg/algea**
- **Close PG** before running this script.
- Run as Administrator to fully clean traces (auto).
- This script **only deletes local data**; it **CANNOT** unban your account.
- False positives may occur with antivirus due to registry and file deletion.
